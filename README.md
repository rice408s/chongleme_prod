# 冲了么 - 技术支持

## 关于应用

冲了么是一款简洁的习惯追踪应用，帮助用户建立自律生活。

## 功能特点
- 📝 每日打卡记录
- 📊 连续天数统计
- 🏆 成就解锁系统
- 📱 简洁优雅界面
- 🔒 本地数据存储，保护隐私

## 需要帮助？

如果你在使用过程中遇到问题，请：

1. 查看下方常见问题
2. 如果问题仍未解决，请[提交Issue](https://github.com/rice408s/chongleme_prod/issues/new)

## 常见问题

### Q: 数据会上传到服务器吗？
A: 不会。所有数据完全存储在本地设备，保护您的隐私。

### Q: 如何备份数据？
A: 数据会随iCloud自动备份，换设备时会同步。

### Q: 应用卡顿怎么办？
A: 尝试重启应用或重启设备。如仍有问题请反馈。

### Q: 如何联系开发者？
A: 请通过GitHub Issues反馈问题。
