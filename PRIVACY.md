# 冲了么 - 隐私政策

最后更新：2026年1月

## 概述

冲了么("我们"或"应用")非常重视用户隐私。本隐私政策说明我们如何收集、使用和保护您的信息。

## 数据收集

### 我们不收集的数据
- **个人身份信息**：我们不收集您的姓名、电子邮件、电话号码等个人信息
- **位置信息**：我们不访问或收集您的位置数据
- **设备信息**：我们不收集您的设备标识符或硬件信息
- **使用分析**：我们不使用任何分析工具跟踪您的应用使用情况

### 本地数据存储
- 所有打卡记录完全存储在您的设备本地
- 数据通过iOS系统的iCloud自动备份（如果您启用了iCloud）
- 我们无法访问您的任何数据

## 数据使用

- **本地功能**：您的数据仅用于在应用内显示统计信息和历史记录
- **无服务器传输**：没有任何数据会发送到我们的服务器或第三方服务器
- **无广告**：我们不使用您的数据进行广告投放

## 数据共享

我们不与任何第三方共享您的数据，因为我们根本无法访问您的数据。

## 数据安全

- 所有数据存储在iOS的安全沙盒环境中
- 数据随设备锁定而加密
- 只有您本人可以访问您的数据

## 您的权利

由于所有数据都在您的设备本地，您拥有完全控制权：
- **删除数据**：卸载应用即可删除所有本地数据
- **导出数据**：可通过应用内分享功能导出成绩卡片
- **控制备份**：通过iOS设置控制iCloud备份

## 第三方服务

本应用不集成任何第三方SDK或服务，包括但不限于：
- 广告网络
- 数据分析工具
- 社交媒体平台
- 云存储服务（除iOS系统级iCloud）

## 儿童隐私

本应用适合所有年龄段用户使用。由于我们不收集任何个人信息，因此不存在儿童隐私问题。

## 政策更新

如果我们更新此隐私政策，会在应用更新说明和本页面中通知您。

## 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：
- GitHub Issues：[提交问题](https://github.com/rice408s/chongleme_prod/issues)

本隐私政策体现了我们对用户隐私的承诺：零数据收集，完全本地存储，绝对隐私保护。
